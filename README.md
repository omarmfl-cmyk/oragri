# 🌿 OrAgri

Plateforme de gestion agricole intelligente — IPM, Culture, Irrigation, Fertilisation, Économie.

## Structure

```
oragri/
├── index.html          ← Page de chargement (loading)
├── pages/
│   ├── selection.html  ← Page de sélection de module
│   └── culture.html    ← Module Culture
├── assets/
│   ├── css/style.css
│   ├── js/app.js
│   └── images/         ← logo.png ici
├── data/
│   └── cultures.json
└── README.md
```

## Stack
- HTML / CSS / JS (statique)
- Netlify (hébergement)
- GitHub Actions (automatisation)
- Open-Meteo API (météo gratuite)
- EmailJS (notifications)

## Langues
- Français
- Darija (العربية المغربية)
